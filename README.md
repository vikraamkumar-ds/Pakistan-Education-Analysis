## 📌 The Central Finding

**50.3% of rural schools in Pakistan are non-functional.**

That's not a rounding error. Out of 6,095 schools surveyed across 7 provinces and 151 districts:

| Status | Count | Share |
|---|---|---|
| ✅ Functional | 3,028 | 49.7% |
| ❌ Non-functional | 3,067 | 50.3% |
| — Open but not operating | 914 | 15.0% |
| — Closed temporarily | 967 | 15.9% |
| — Closed permanently | 1,186 | 19.5% |

### Gender Gap

Girls-only schools have an **8 percentage point higher** non-functional rate than boys-only schools — consistently across every province.

| School Type | Non-Functional Rate |
|---|---|
| Boys-Only | ~47% |
| Girls-Only | ~55% |
| Gap | ~8pp |

## 🤖 Machine Learning Model

### Goal
Predict whether a school is at high risk of being non-functional using **infrastructure features only** — no attendance data required. This allows policymakers to flag at-risk schools without visiting every location.

### Important: Data Leakage Fix
The original dataset included `S007P` (students present at survey time). Including this gave an artificially high AUC of 0.877 — but it's a leakage variable: a non-functional school will naturally have zero students present. Removing it gives an honest, generalizable model.

### Performance

| Metric | Value |
|---|---|
| Algorithm | Random Forest Classifier |
| Accuracy | 71.0% |
| F1 Score | 0.727 |
| ROC-AUC | 0.771 |
| Training split | 80% / 20% |

### Features Used

| Feature | Description |
|---|---|
| `province_encoded` | Province (strongest predictor) |
| `S03a1` | Teacher present at survey time |
| `S03d1` | Drinking water available |
| `S03e1` | Toilet facility available |
| `S03f1` | Electricity available |
| `is_girls_school` | School is girls-only |
| `is_boys_school` | School is boys-only |
| `is_primary` | Primary level school |
| `is_middle` | Middle/higher level school |

## 🗺️ Known Issue — GeoJSON

The included `pakistan_districts.geojson` contains **32 divisions**, not 151 districts. The choropleth map currently renders at division level as a workaround. To enable full district-level mapping, replace the file with a district-level GeoJSON from:
- [GADM Pakistan](https://gadm.org/download_country.html)
- [Humanitarian Data Exchange](https://data.humdata.org)

## 📋 Recommendations

### Policy
- 🔴 **Emergency rehabilitation** in Gilgit-Baltistan and AJK (both >66% non-functional)
- 🟠 **Punjab audit** — 62.5% rate in the most populous province = largest absolute number of affected children
- 👧 **Girls school equity fund** — dedicated stream to close the 8pp gender gap
- ⚡ **Electrify first** — electricity is the scarcest resource with compounding benefits
- 👨‍🏫 **Teacher attendance reform** — strongest single predictor of school functionality

### Technical
- Replace GeoJSON with a district-level file for full 151-district map
- Run `03_ML_Model.ipynb` to regenerate SHAP and confusion matrix visuals
- Load `district_summary.csv` in Tab 3 to enrich district metrics
