# 🏫 Pakistan Rural Education Dashboard
### ASER 2023 — School Infrastructure & Non-Functional School Predictor

An interactive data science dashboard analyzing rural school infrastructure across **7 provinces and 151 districts of Pakistan**, built on the [ASER 2023](http://www.aserpakistan.org/) school survey dataset.

---

## 📊 What This Project Does

- **Explores** infrastructure gaps across 6,095 rural Pakistani schools
- **Visualizes** the gender gap in school functionality (boys vs. girls schools)
- **Maps** non-functional school rates by province and district
- **Predicts** whether a school is at high risk of being non-functional using a Random Forest ML model trained on infrastructure features

> **Key Finding:** Over 50% of rural Pakistani schools are non-functional — meaning they are either open but broken, temporarily closed, or permanently shut.

## 🚀 Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/your-username/pakistan-education-dashboard.git
cd pakistan-education-dashboard
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv
source venv/bin/activate        # Linux/macOS
venv\Scripts\activate           # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the dashboard

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`

## 🤖 ML Model Details

| Property | Value |
|---|---|
| Algorithm | Random Forest Classifier |
| Training data | 6,095 schools (80/20 train/test split) |
| Target variable | `nonfunctional` (binary: 0=functional, 1=non-functional) |
| Accuracy | 71.0% |
| F1 Score | 0.727 |
| ROC-AUC | 0.771 |
| Features | 10 (see below) |

### Features Used

| Feature | Description |
|---|---|
| `province_encoded` | Label-encoded province (strongest predictor) |
| `S03a1` | Teacher present at survey time |
| `S03d1` | Drinking water available |
| `S03e1` | Toilet facility available |
| `S03f1` | Electricity available |
| `is_girls_school` | School serves girls only |
| `is_boys_school` | School serves boys only |
| `is_mixed_school` | Co-educational school |
| `is_primary` | Primary level school |
| `is_middle` | Middle/Higher Secondary school |

> ⚠️ **Note on data leakage:** `S007P` (student presence/attendance) was deliberately excluded. A non-functional school naturally has low attendance — including it would inflate model accuracy without adding genuine predictive value.

## ⚠️ Known Issues & Fixes

### 1. 🗺️ Choropleth Map Shows Nothing
**Root cause:** The included `pakistan_districts.geojson` contains **32 administrative division polygons**, not the 151 district-level polygons needed.

**Fix:**
1. Download the correct district-level GeoJSON from:
   - [GADM Pakistan Level-3](https://gadm.org/download_country.html)
   - [HDX Pakistan Admin Boundaries](https://data.humdata.org/dataset/cod-ab-pak)
2. Place it in `data/pakistan_districts.geojson`
3. Update `featureidkey` in `app.py` to match the property name in your new file (e.g., `properties.DISTRICT` or `properties.ADM3_EN`)

### 2. 📦 Model Requires `shap` Package
The `best_model_bundle.pkl` was saved with a SHAP explainer object. The `shap` library must be installed for the pkl to load. It is included in `requirements.txt`.

### 3. ⚙️ scikit-learn Version Warning
The model was trained with scikit-learn 1.7.2. If your environment uses a different version, you may see an `InconsistentVersionWarning`. This is non-critical but can be resolved by re-training and re-saving the model in your deployment environment.

```bash
# To suppress the warning
import warnings
from sklearn.exceptions import InconsistentVersionWarning
warnings.filterwarnings("ignore", category=InconsistentVersionWarning)
```

### 4. 🖼️ SHAP Plots Missing
The SHAP visualizations (`visuals/shap_bar.png`, `visuals/shap_beeswarm.png`) are generated by the notebook. Run notebook `03_ML_Model.ipynb` to generate them before launching the app.

## 📁 Data Description

### `aser_2023_clean.csv` — Main Dataset

| Column | Type | Description |
|---|---|---|
| `RNAME` | string | Province name |
| `DNAME` | string | District name |
| `STYPE` | int | School type (1=Primary, 2=Middle/Higher) |
| `S001` | int | Area type (1=Urban, 2=Rural, 3=Peri-urban) |
| `S002` | float | School gender (1=Boys, 2=Girls, 3=Mixed) |
| `S00` | int | School status (1=Open/functional, 2=Open/non-func, 3=Closed temp, 4=Closed perm) |
| `S03a1` | float | Teacher present (1=Yes, 0=No) |
| `S03d1` | float | Drinking water (1=Yes, 0=No) |
| `S03e1` | float | Toilet (1=Yes, 0=No) |
| `S03f1` | float | Electricity (1=Yes, 0=No) |
| `school_gender` | string | Readable gender label |
| `nonfunctional` | int | **Target variable** (0=functional, 1=non-functional) |

### `district_summary.csv` — Aggregated by District

| Column | Description |
|---|---|
| `RNAME` | Province name |
| `DNAME` | District name |
| `total_schools` | Number of schools surveyed |
| `nonfunctional_count` | Number of non-functional schools |
| `nonfunctional_rate` | Rate as decimal (0–1) |
| `district_clean` | Lowercase district name for GeoJSON matching |
